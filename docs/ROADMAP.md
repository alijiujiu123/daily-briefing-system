## Phase 1: MVP (Week 1-2)

### 目标

### 任务清单

**RSS 抓取**
- [ ] 实现 RSS 解析器
- [ ] 支持并发抓取
- [ ] 错误处理和重试机制
- [ ] 文章去重（基于 URL）
- [ ] 增量更新（只抓取新文章）

**数据处理**
- [ ] 数据库 schema 设计
- [ ] SQLite 初始化脚本
- [ ] 文章存储
- [ ] 查询和过滤

**简报生成**
- [ ] 模板系统
- [ ] Markdown 格式化
- [ ] 分类逻辑（AI/创业/安全等）

**Telegram 推送**
- [ ] Bot 集成
- [ ] 消息格式化
- [ ] 发送逻辑

**调度系统**
- [ ] Cron 任务配置
- [ ] 时区处理
- [ ] 错误告警

### 交付标准
- 每天 8:00 自动抓取并推送
- 包含至少 10 篇文章
- Telegram Bot 正常工作
